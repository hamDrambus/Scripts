FAQ Windows NT/2K

Как сделать, чтобы на русской раскладке клавиатуры на цифровой клавиатуре вместо запятой печаталась точка (кнопка Del).

Отвечает D.A.E.M.O.N.:
Установить в свойствах "Язык и стандарты" соответствующий разделитель целой и дробной части числа.

Отвечает Вячеслав:
dll-ку прямо сейчас не найду :( Метода была такой: берется файл kdbru.dll, любым hex-редактором меняем 2E на 2C по адресу 08С4, кладем исправленный файл поверх всех таких файлов в каталоге Windows (в т.ч. и в DllCache). Если сильно хочется сохранить оригинальную dll, можно отредактированную переименовать и прописать ее имя по адресу HKEY_LOCAL_MACHINE\SYSTEM\ CurrentControlSet\Control\Keyboard Layouts\00000419 Layout File P.S. То же самое делал редактором раскладок. По непонятной причине получаемый kbdru.dll становится на 2k больше - поэтому решил сделать "ручками" :) Удачи!

Отвечает Иван:
Можно воспользоваться любой программкой переназначения кнопок клавиатуры.
